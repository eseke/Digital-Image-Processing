# Digital-Image-Processing
Digital Image Processing course homeworks

Homework 1:
 - [Gamma correction](https://en.wikipedia.org/wiki/Gamma_correction) and various methods for hdr to sdr conversion
 - Selective decolorization of image
 - Implementation of [Bilateral filter](https://en.wikipedia.org/wiki/Bilateral_filter)

Homework 2:
 - Inverse [halftoning](https://en.wikipedia.org/wiki/Halftone)
 - Removing [additive Gaussian noise](https://en.wikipedia.org/wiki/Additive_white_Gaussian_noise) with Histogram of local variances method
 - Implementation of [Wiener filter](https://en.wikipedia.org/wiki/Wiener_filter) on image blurred with Gaussian filter
 - Implementation of Gaussian filter in spatial and frequency domain

Homework 3:
 - Segmentation of traffic lines from dashcam video
 - Implementation of [Canny edge detector](https://en.wikipedia.org/wiki/Canny_edge_detector)
 - Recognition of traffic lane position and direction in dashcam video
